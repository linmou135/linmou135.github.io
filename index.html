如何开通博客

根据教程https://www.cnblogs.com/chenlove/p/15058170.html所写，大家可以去看看哦
首先第一步，先下载所需要的东西，像git啊，像node.js啊这些都是得进行安装的
然后下载完成了之后可以用git –version来查看版本，如果有显示出git版本号，说明已经成功下载了
相同的，我们可以使用node -v来看node的版本号，如果有就说明安装完成,这些活动都是在git bash完成的
这些环境都安装成功以后，我们可以在git bash里面输入npm install hexo -g来下载hexo
然后要测试是否安装完成可以用hexo -v来查看，如果显示出版本号的话就意味着安装成功
然后可以开始本地配置ssh key了
使用ssh-keygen-t rsa -C”邮箱地址”//你自己使用来注册git的哪个我的是qq邮箱
连续回车
就能生成rsa文件了
一般来说都在用户然后你用户名.ssh文件夹里，哦当然你如果以前有用过那就不知道了
然后我们能发现一个后缀是pub的文件，用记事本点开这个文件然后全部都复制一遍离开
来到github，点开你个人主页卡就有个小箭头能展开的，可以发现有个setting，点击然后就能看到一个
ssh and gpg keys了，把复制的东西粘贴到key里，名字随便取点add就行了
可以在git bash里面用ssh -T git@github.com来看看能不能连接到github，能连接进行下一步操作
配置git config –global user.name “”//引号填自己的git用户名
git config –global user.email “”//引号填自己的git注册名
接下来就很简单的搭博客环节，好像只要四个命令？
第一个是先博客初始化，hexo init
然后生成静态网页 hexo g
然后你可以先小预览下什么样子的
用hexo s
然后网址输入http：//localhost:4000，当然，这好像是临时的，你搭完博客后你会发现进不去了
先搭建一个git项目，respository ，名字是
你git名字.github.io
记得勾选add a reanme file
然后创造
编辑_config.yml,博客目录下的，而不是themes的landscape里的，别弄错，应该也不会弄错吧
最后一段改成这个
deploy:
type: git
repository: 项目的code那边的ssh
branch: main//这个很重要，卡我很久，你看看你那边的那个文件是main还是master从而改成和他一样
接下来，开杀，使用hexo d上传了？
那你大概率会得到报错
因为我们要先安装插件才行npm install –save hexo-deployed-git
然后再上传就不会有问题了
hexo d直接使用，得到我们的博客，是不是很简单？
接下来使用hexo new ‘自己取名-first-blog’就可以创建新博客了
我们可以去source里面取寻找你的那个.md文件进行写字就可以写出博客了
